## Submit Your Project

**Want your project featured?**

### Requirements

Your project should:
- Use Openwater platforms (OpenLIFU or OpenMOTION)
- Be functional (at least MVP/prototype stage)
- Have clear documentation
- Include source code or build instructions
- Follow open-source principles

### How to Submit

1. **Fork** this repository
2. **Add your project** to this file following the template below
3. **Include:**
   - Project name and description
   - Links (GitHub, website, paper)
   - Images or screenshots
   - Your name/affiliation
4. **Submit** a pull request

### Project Template


## Example Project (Template)

### Acoustic Field Optimizer

**By:** Jane Researcher / Stanford University  
**Date:** December 2025  
**Platform:** OpenLIFU  
**Category:** Technical Extension

**Description:**  
An optimization tool that automatically adjusts ultrasound parameters to maximize acoustic pressure at target locations while minimizing skull heating. Uses gradient descent algorithms for real-time parameter tuning.

**Key Innovation:**  
First open-source tool for automated ultrasound parameter optimization with skull heating constraints.

**Links:**
- GitHub: https://github.com/username/acoustic-optimizer
- Paper: https://doi.org/10.xxxx/example
- Demo: https://youtube.com/watch?v=example

**Images:**  
![Optimization Interface](https://example.com/screenshot.png)

**Technologies Used:**
- OpenLIFU-python v1.0
- SciPy optimization
- NumPy, Matplotlib
- K-Wave acoustic simulation

**Impact:**  
Reduces treatment planning time from hours to minutes. Improves safety by automatically enforcing thermal constraints.

**Status:** Beta (Available for testing)
