## Integration Tools

### Medical Imaging

**3D Slicer**
- Website: https://www.slicer.org
- Purpose: Medical image visualization and processing
- Integration: SlicerOpenLIFU extension
- License: BSD-style

**SimpleITK**
- Website: https://simpleitk.org
- Purpose: Medical image registration and segmentation
- Use with: OpenLIFU for image processing
- License: Apache 2.0

**PyDICOM**
- Website: https://pydicom.github.io
- Purpose: DICOM file handling
- Use with: Both platforms for medical image I/O
- License: MIT

### Acoustic Simulation

**K-Wave**
- Website: http://www.k-wave.org
- Purpose: Acoustic and ultrasound simulation
- Use with: OpenLIFU for treatment planning
- License: LGPL

**FOCUS**
- Website: https://www.egr.msu.edu/~fultras-web/
- Purpose: Fast Object-oriented C++ Ultrasound Simulator
- Use with: OpenLIFU simulations
- License: Open-source
