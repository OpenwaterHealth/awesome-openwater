## Contributing

**Have a paper using Openwater?**

Add it to the appropriate list:
1. Choose: [openlifu-papers.md](openlifu-papers.md) or [openmotion-papers.md](openmotion-papers.md)
2. Follow the format:
   ```markdown
   **Title** (Year)
   - Authors: [Names]
   - Journal: [Journal name and citation]
   - DOI: [link]
   - Platform: OpenLIFU/OpenMOTION
   - Summary: [1-2 sentence summary]
   ```
3. Submit a pull request
