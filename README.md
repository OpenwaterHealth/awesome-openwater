## Platforms

### OpenLIFU (Low Intensity Focused Ultrasound)

- [OpenLIFU-python](https://github.com/OpenwaterHealth/OpenLIFU-python) - Core Python toolbox for ultrasound planning and control
- [SlicerOpenLIFU](https://github.com/OpenwaterHealth/SlicerOpenLIFU) - 3D Slicer extension for treatment planning
- [OpenLIFU-3DScanner](https://github.com/OpenwaterHealth/OpenLIFU-3DScanner) - Photogrammetry application for transducer localization

**Applications:**
- Neuromodulation and brain stimulation
- Glioblastoma treatment
- Stroke therapy
- Non-invasive brain surgery

### OpenMOTION (Motion Tracking & Blood Flow Imaging)

- [OpenMOTION-Pylib](https://github.com/OpenwaterHealth/OpenMOTION-Pylib) - Python library for motion data acquisition
- [OpenMOTION-TestAPP](https://github.com/OpenwaterHealth/OpenMOTION-TestAPP) - Testing and demo application

**Applications:**
- Stroke diagnosis
- Cardiovascular monitoring
- Blood flow quantification
- Motion tracking

## Contributing

We welcome contributions to this awesome list!

### How to Contribute

1. **Fork** this repository
2. **Add** your resource to the appropriate section
3. **Follow** the format of existing entries
4. **Submit** a pull request

### Contribution Guidelines

- Only add resources you've personally used or verified
- Ensure resource is relevant to Openwater development
- Keep descriptions concise and accurate
- Check that links work
- Follow alphabetical ordering within sections

See [contributing.md](contributing.md) for detailed guidelines.

### What to Add

**Good additions:**
- Tutorials you've written
- Research papers using Openwater
- Tools and libraries you've developed
- Community projects you've built
- Conference talks you've given
- Resources you've found helpful

**Please don't add:**
- Resources not related to Openwater
- Commercial products without clear value
- Unverified or broken links
- Duplicate entries

## About Openwater

Openwater is building open-source medical devices to democratize healthcare technology and bring advanced medical imaging and therapeutic systems to 3.5 billion underserved people worldwide.

**Mission:** Reduce medical device development costs from $119M to $15M through open-source collaboration.

**Vision:** Become the "Linux of medical devices" - a global platform enabling hundreds of developers to innovate.

**Learn more:** [openwater.health](https://openwater.health)
